# Kolam Generator

The **Kolam Generator** is the core component for creating cryptographic credentials based on Kolam art patterns.

## Features

*   **Parameter Configuration**: Allows users to set Grid Size (k), Randomness (m), Symmetry, and other visual parameters.
*   **Generation & Analysis**: Generates a unique Kolam matrix and performs cryptographic analysis to extract keys (Pure, CSPRNG, Hybrid).
*   **Preview**: Visualizes the generated pattern in real-time.
*   **Credential Export**: Exports the generated data as a JSON file or a PNG image with embedded metadata, which serves as the "key" for the Secure Chat.

## Usage

1.  Adjust parameters (e.g., increase Grid Size for higher security).
2.  Click **"Generate & Analyze"**.
3.  Download the resulting credential to use in the chat or save to history.
