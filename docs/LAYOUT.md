# Layout

**Layout** determines the visual presentation of the Kolam grid.

## Options

*   **Square Grid (no rotate)**: The standard presentation where cells are arranged in an orthogonal grid (rows and columns).
*   **Diamond Grid (rotate 45°)**: The entire grid is rotated by 45 degrees. This is a traditional way of drawing Kolams (Pulli Kolam) where the dots are arranged in a diamond pattern.

## Impact
*   **Visual**: Changes how the pattern looks to the human eye.
*   **Data**: Does **not** change the underlying matrix data or the generated keys. The logical structure (connections between nodes) remains the same; only the rendering coordinate system changes.
