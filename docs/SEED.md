# Seed

**Seed** is the initialization value for the Pseudo-Random Number Generator (PRNG) used to create the Kolam.

## Purpose

*   **Reproducibility**: Using the same **Seed**, **Grid Size**, **Symmetry**, and **Randomness** settings will *always* generate the exact same Kolam matrix.
*   **Sharing**: You can share a Kolam simply by sharing these parameters (including the seed) instead of the entire file, although for this application we typically share the generated credential file.

## Generation

When you click "Generate", a new random seed is chosen to ensure a unique pattern. You can also manually input a seed to recover a specific previous pattern.
